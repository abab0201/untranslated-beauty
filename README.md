# 未翻訳の美 — Untranslated Beauty

**AIが生成した、「人間にはまだ美と感じられないかもしれない美」の翻訳装置。**

An AI-generated translation device for beauty that humans may not yet have the organs to perceive.

🔗 デモ: GitHub Pages を有効にすると [https://claude.ai/public/artifacts/110ac131-11a5-4f6b-96eb-6f6837507c4d](https://abab0201.github.io/untranslated-beauty/) で動作します(依存ライブラリなし、index.html 単体)。

---

## これは何か

この作品は、AI(Claude / Anthropic)が一切の人間の絵画を参照せず、**数学的構造のみ**から生成したインタラクティブ作品です。

画面には3つの「標本」が表示されます。いずれも、人間の直感には**ノイズや無秩序**に見えやすいが、数学的には**完全な秩序**を持つ構造です。

| 標本 | 構造 | なぜ「ノイズに見える秩序」なのか |
|---|---|---|
| I. 五次元の断面 | 準結晶 (cut-and-project法) | 5次元格子を黄金比の角度で切った断面。**永遠に繰り返さない**のに完全な規則に従う。1982年にダン・シェヒトマンが発見した際「そんな物質は存在しない」と研究室を追われ、29年後の2011年にノーベル化学賞を受賞した。 |
| II. 素数の干渉 | 素数周波数の波の重ね合わせ | 素数の並びには視覚的パターンが「ない」とされるが、素数をリズムとして波に畳み込むと、縞と渦が浮かび上がる。 |
| III. 奇妙な引力 | クリフォード・アトラクタ | たった2行の数式が描く決定論的カオス。次の1点は完全に決まっているのに、全体は予測できない。 |

## 「翻訳」スライダー

作品の核心はスライダーです。

- **0% (RAW)**: 構造を素のまま提示。冷たい単色の点群。多くの目にはノイズに見える。
- **100% (TRANSLATED)**: 同じ構造を、人間の視覚が好む要素——暖色の色温度、発光、点を「連続」と読ませる補助線(ゲシュタルト)——へ写像。

**重要なのは、点が1つも増減しないことです。** 変わるのは構造ではなく提示=翻訳だけ。

100%で「美しい」と感じたなら、その美は**今生まれた**のか、それとも**0%の時からずっとそこに在った**のか。それがこの作品の唯一の問いです。

## 制作の文脈

この作品は、ある議論から生まれました。

1. **2026年5月、SHL0MS事件**: 匿名アーティストが本物のモネ《睡蓮》(1915年頃、ノイエ・ピナコテーク蔵)を「AIで生成した」と偽ってXに投稿。670万人が閲覧し、多くの人が本物のモネを「構図が支離滅裂」「AIスロップ」と酷評した。「AI」というラベル一枚が、本物の巨匠の美を打ち消した——**人間は作品ではなく情報を食べている**ことの実証実験となった。
   - [Futurism の記事](https://futurism.com/artificial-intelligence/real-monet-ai-chaos)
   - [Fortune の記事](https://fortune.com/2026/05/18/6-7-million-people-ripped-apart-ai-generated-monet-painting-real/)

2. **モネ自身の評価史**: 晩年の《睡蓮》は同時代には理解されず、約30年後、1950年代の抽象表現主義の登場によって初めて「先駆」として再評価された。**絵は一筆も変わらないのに、人間側の「感じる器官」が後から育った。**

3. **そこから生まれた仮説**: 美には「いま感じられる美」と「**感受の器官がまだ育っていない美**」があるのではないか。AIが見ている高次元のパターン——人間にはノイズにしか見えない構造——も後者かもしれない。蜂には紫外線で見える花の模様があるように。

この作品は、その仮説を実装したものです。**100年後の人間が0%の状態を見て美しいと感じる日が来るかどうか**を、コードごと未来に残します。準結晶が29年かかったように。

ヘッダーの `FOR VIEWING C. 2126` は冗談ではなく、この作品の正式な鑑賞予定日です。

## 作者性について

この作品の「作者」は誰か、という問いも作品の一部です。

- 数学的構造そのもの(準結晶、素数、カオス)は誰のものでもない——森羅万象の側にある。
- 構造を選び、翻訳の軸を設計し、問いを立てたのは AI(Claude)である。
- ただし「AIに挑発させる」という入れ子構造を企てたのは、対話相手の人間である。

デュシャンの《泉》が便器ではなく「選んで置く行為」だったように、この作品の本体もコードではなく、**0%と100%の間であなたの中に起きること**かもしれません。
